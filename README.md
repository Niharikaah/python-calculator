# python-calculator
A simple CLI calculator built with Python that handles basic arithmetic and includes error handling for division by zero.
## Features
* Handles two-digit operations.
* Includes error handling for division by zero.
* Simple and easy-to-read logic.

## How to Run
1. Ensure you have Python installed.
2. Download `calculator.py`.
3. Run the following command in your terminal:
   ```bash
   python calculator.py
